Projeto: Sistema de Gestão para Óticas (PWA)

O Sistema de Gestão para Óticas é uma aplicação web completa e responsiva, desenvolvida para otimizar as operações diárias de óticas de pequeno e médio porte. Utilizando uma arquitetura moderna de PWA (Progressive Web App), o sistema oferece uma experiência de usuário fluida e intuitiva, combinando a praticidade de um site com a funcionalidade de um aplicativo nativo. Ele foi projetado para centralizar a gestão de clientes, receitas, agendamentos, estoque e vendas, fornecendo uma solução robusta e segura.

Sistema de Reservas Online Para Bares e Baladas

O projeto é uma aplicação web full-stack, desenvolvida para modernizar a gestão de reservas de um bar/balada. O sistema integra um frontend intuitivo para clientes e um painel de controle seguro para administradores, eliminando ineficiências operacionais e a dependência de métodos de comunicação manuais, como o WhatsApp.
